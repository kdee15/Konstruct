Hello there!

Welcome to the Konstruct documentation. This is where you will pick up all you need to know to get Konstruct up and running quickly.

What is Konstruct?

Although Konstruct is developed to be as unobtrusive and simple as possible, there are a few code practises & patterns included. These are things we use daily, and I felt it was a good enough way to do things for core. Of course you are welcome to change that, and even make a pull request to get your way in.
