Get started with Konstruct!

We highly reccommend that you get started with the Konstruct Boilerplate, which is a great platform to start your project form. Out of the box, the boilerplate hooks you up with Bower for component management, Gulp for task management and an awesome starter kit so you can begin writing your awesome site / app / rocket in seconds!

Before you begin:

The boilerplate assumes that you have Nodejs & Bower already installed. If you don't please go get that done first!

1. Get the Konstruct Boilerplate.

The simplest way to get hold of the boilerplate is to clone the project from Github:

git clone https://github.com/traaidmark/konstruct-boilerplate.git [YOUR_FOLDER]

2. Install Gulpjs.

Next we need to get GulpJS up and running. If you have NodeJS & Gulp installed, this should be as simple as running:

npm install

3. Initiate your new site.

The Konstruct Boilerplate relies on Gulp to do all of the site related tasks, and this includes the Bower components. You are welcome to run the bower commands yourself, but we prefer if you follow the method below:

gulp init

This command will run bower install, and move the Konstruct library to a useable directory, and then do a clean sweep, to remove the .git folder and other small errant files. This will leave you with a clean site setup to start working with!

5. Go Build Something Awesome!

You are ready to start building!

Be sure to check out the Konstruct Gulp Documentation for a list of all the available commands and their usage.
